## 💻 How to Run the Programs

1. Install a C compiler (like GCC)
2. Open terminal in the project folder
3. Compile the file:

```bash
gcc filename.c -o output
```

4. Run the program:

```bash
./output
```

*(On Windows use `output.exe` instead of `./output`)*
